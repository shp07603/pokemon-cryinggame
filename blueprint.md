# PokeCry: Pokemon Cry Quiz Game

## Overview
A retro-styled web game where players listen to Pokemon cries and guess which Pokemon they belong to. The game features a CRT-inspired aesthetic with pixel art and modern CSS effects.

## Technical Stack
- **Frontend:** Vanilla HTML5, CSS3 (Baseline features), Modern JavaScript (ES Modules).
- **API:** [PokeAPI](https://pokeapi.co/) for Pokemon data, sprites, and cries.
- **Assets:** Google Fonts (Press Start 2P, Noto Sans KR).
